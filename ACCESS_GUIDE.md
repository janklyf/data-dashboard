## 🔍 页面功能

### 1. 数据块管理
- ✅ 点击"添加数据块"按钮打开模态框
- ✅ 填写标题、内容、分组
- ✅ 选择尺寸（小/中/大）和颜色（灰/蓝/绿/黄/红）
- ✅ 点击"保存"添加数据块

### 2. 分组管理
- ✅ 点击"添加分组"按钮打开模态框
- ✅ 输入分组名称
- ✅ 点击"保存"添加分组

### 3. 数据块操作
- ✅ 点击"编辑"按钮修改数据块
- ✅ 点击"删除"按钮删除数据块
- ✅ 使用筛选下拉框按分组查看数据块

## 🛠️ 测试 API

### 获取分组列表
```bash
curl http://localhost:8000/api.php/groups
```

### 获取数据块列表
```bash
curl http://localhost:8000/api.php/blocks
```

### 添加数据块
```bash
curl -X POST http://localhost:8000/api.php/blocks \
  -H "Content-Type: application/json" \
  -d '{"title":"测试标题","content":"测试内容","group_id":1,"size":"small","color":"blue"}'
```
