# 表单页面说明

## 已集成到 index.php

数据块和分组的表单功能已集成到主页面中，通过 Bootstrap 模态框实现。

### 数据块表单功能

**位置**: `index.php` 中的 `#blockModal`

**字段**:
- 标题（必填）
- 内容（必填）
- 分组（必填）
- 尺寸（小/中/大）
- 颜色（灰/蓝/绿/黄/红）

**功能**:
- 添加数据块
- 编辑数据块
- 自动加载分组列表
- AJAX 提交数据

### 分组表单功能

**位置**: `index.php` 中的 `#groupModal`

**字段**:
- 分组名称（必填）

**功能**:
- 添加分组
- 编辑分组（需要单独 API 支持）
